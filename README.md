## 💻 Featured Projects

### 📚 [ReaDirect](https://github.com/kazamijimin/ReaDirect)

> An educational reading support system designed to help learners through structured reading activities, assessment flow, learner progress tracking, and AI-assisted reading interaction.

**Project Focus:**

* Reading support system for learners
* Laravel API backend
* React Native + Expo frontend
* Progressive Web App support
* PostgreSQL database
* Structured repository for backend, frontend, database, documentation, infrastructure, and scripts

**Tech Stack:** Laravel, React Native, Expo, PostgreSQL, REST API, PWA

---

### 🧠 [ReaDirect AI-ASR](https://github.com/kazamijimin/ReaDirect-AI-ASR)

> A supporting AI speech recognition service for ReaDirect, focused on processing learner audio responses for pronunciation and reading activity evaluation.

**Project Focus:**

* AI-assisted speech recognition
* Learner voice response processing
* Reading and pronunciation support
* Integration with the main ReaDirect system

**Tech Stack:** Python, FastAPI, ASR, AI Integration

---

### 📊 [Population Dynamic System](https://github.com/kazamijimin/population-dynamic-system)

> A full-stack web application for managing and simulating population dynamics with dashboards, analytics, inventory management, and role-based access control.

**Project Focus:**

* Population simulation and modeling
* Admin and manager dashboards
* Inventory and resource management
* Analytics and report generation
* Role-based authentication and authorization

**Tech Stack:** Django, Django REST Framework, PostgreSQL, Neon DB, React, Vite, Axios, React Router

---

### 🩺 MediChecker

> A health-related checker system designed to organize medical or wellness-related information into a clearer and more accessible digital workflow.

**Project Focus:**

* Health information checking workflow
* User-friendly interface
* Organized data handling
* Simple and practical system design
